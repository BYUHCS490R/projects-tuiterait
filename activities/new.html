Tu'iterai Tunutu

Internet

The internet is optic cables that connect all of our machines using protocols

Web

The web is a way to share information using the internet

Internet vs Web

The internet is like a system of road that connects us and the web is like cars or building that transports and store information.

ASCII and Unicode

Those are like the alphabet but in computer language, ASCII is the older version so its mostly designed for the english alphabet while Unicode is a little more new that includes most character from all the worlds languages like symboles and scripts.

UFT-8

It's a 8-bit standar character encoding method that makes ASCII and Unicode compatible

URL

Short for Uniform Resource Locator that uses a unique address to find the location of a resource on the internet like in our road we need to find the building and URL is the identifier for said building.

Domain name

Domain name is a URL but human friendly because it translates easy to remember names into numberical IP addresses.

HTTP

Short for Hypertext Transfer protocol which allows web browser to request and receive data. This transfer of data permit us to see the content of a webpage.

HTTPS

Basically the same as HTTP but secure because it encrypt our data and ensure that our information is private and secure.

IP Address

Like a unique address for our devise that for nentwork to allow to identify and communicate with us.

DNS

It's the protocol that translate domain names into IP addresses.

W3C Recommendations

It's a web standard that defines how web technologies should work.

XML, XHTML, HTML, HTML5

These are all languages used to create web pages but they are all different in their own way. HTML is the fundamental langugage and HTML5 is the latest version for it, HTML is mostly used to structure content and tell a webbrowser how to display it like a word document with instructions, HTML is relatively flexible with its syntax. XML is more focused on displaying data and transporting data. It's more stricter about syntax than HTML. XHTML is like the cross over between HTML and XML
