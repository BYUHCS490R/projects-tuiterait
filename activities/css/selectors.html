1. You can select by tag name (p), by ID using #idname, by class using .classname, by children or descendants (main p), and you can group selectors with commas.

2. An ID is for one unique element and uses #. A class can be used on many elements and uses ..

3. SPAN is inline and only changes part of a line. DIV is block and takes the whole width.

4. DIV is just a generic block. HEADER, MAIN, and FOOTER are semantic tags that say what that block means.

5. You can pick colors by name (red), hex (#ff0000), rgb(rgb(255,0,0)), hsl(hsl(0,100%,50%)), or with rgba/hsla for opacity.
